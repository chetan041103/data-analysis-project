Customer Purchase Behavior Analysis

A complete end-to-end Data Analytics project using Python (Data Cleaning), PostgreSQL (Querying), and Power BI (Visualization).

 Project Overview:

This project focuses on understanding customer purchase patterns using a real dataset.
The workflow includes:

Cleaning and preprocessing raw data using Python

Writing analytical queries in PostgreSQL

Building an interactive Power BI dashboard to visualize insights
The goal is to help businesses make data-driven decisions based on customer behavior.
